\# 🔑 Formulário de Login

Este projeto foi desenvolvido para praticar HTML e CSS, criando um formulário de login funcional e visualmente organizado.



\## 🛠️ Tecnologias utilizadas

\- HTML  

\- CSS  



\## 🎯 Objetivo

Praticar a criação de formulários estilizados usando apenas HTML e CSS, aplicando alinhamento, cores e posicionamento de elementos.



\## 🌐 Visualizar projeto

\[Ver demo online](https://liadaine.github.io/formulario-login/)



\## 📚 O que foi praticado

\- Estruturação do formulário em HTML  

\- Estilização com CSS (cores, bordas, fontes e botões)  

\- Alinhamento e organização de campos de entrada (usuário, senha, botão de login)  

\- Criação de layout limpo e funcional
